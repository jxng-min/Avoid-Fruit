# 과일 피하기(Avoid Fruit)

### 개요

게임, 친구/채팅을 접목시킨 친목형 대학교 커뮤니티 서비스인 'SUCAT'에 사용하기 위해 제작 중입니다. 유니티 엔진 5를 활용해 2D 픽셀 플랫포머 게임을 제작하는 것이 목표입니다.
